¡Este es el inicio de una gran aventura!

Todo comienza con Platzi y su increíble curso de Git y GitHub.

Para poder aprender a utilizar el control de versiones, utilizaré html y css para los ejemplos.

Por trabajo tuve que utilizarlo muchas veces, pero los comandos básicos. Hasta ahora he aprendido cosas muy buenas. Un ejemplo de ello es el entorno gráfico con el que puedo ver toda la información de git de mi proyecto, para ello el simple comando gitk, abre un software donde puedo ver toda la historia y hacer muchas cosas con él.

Además de ello, aprendí a hacer una conexión SSH con GitHub para poder conectar de forma remota con seguridad. Lo de las etiquetas, es una forma genial de organizar el proyecto para llevar un orden del desarrollo. Los conflictos ya no me dan miedo, gracias a Freddy puedo fácilmente resolverlos.

GitHub es también una herramienta fantástica para el control de versiones en remoto, ya que permite también añadir diferentes desarrolladores que puedan colaborar en el proyecto. Definitivamente, el control de versiones con git es simplemente genial.
